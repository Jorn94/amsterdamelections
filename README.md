# Amsterdam Election Quiz 2026

A GitHub-ready React quiz based on the uploaded Amsterdam municipal elections 2026 party comparison sheet.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

1. Create a GitHub repo called `amsterdam-election-quiz`
2. Push this project to the `main` branch
3. In GitHub repo settings, go to **Pages**
4. Set **Source** to **GitHub Actions**
5. Push again if needed and wait for the workflow to finish

Your live URL will be:

```text
https://YOUR-USERNAME.github.io/amsterdam-election-quiz/
```
